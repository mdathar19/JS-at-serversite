What is a stream in node JS?
What are Streams? Streams are objects that let you read data from a source or write data to a destination in continuous fashion

What is stream and buffer in Node JS?
Streams work on a concept called buffer. A buffer is a temporary memory that a stream takes to hold some data until it is consumed.

Why we use streams in Nodejs?
Streams are one of the fundamental concepts of Node. js. Streams are a type of data-handling methods and are used to read or write input into output sequentially. Streams are used to handle reading/writing files or exchanging information in an efficient way.
